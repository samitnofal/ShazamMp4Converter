# ShazamMp4Converter

The main purpose of this project is to extract mp3 tracks from mp4 videos. Furthermore, the software will try to shazam the song using ShazamAPI and rename the mp3 track accordingly.

Usage:

Steps
1) Clone the Repository 
2) Run "python convert_Mp3_to_Mp4_shazam_and_rename.py -p <path_to_folder_where_mp4_is_stored>"
